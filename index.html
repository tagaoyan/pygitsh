Install

Before install, you should have

  • Git (you should already have)
  • Python 3.x (with an executable named python3)
  • An account for serving git via SSH (assume it named git)

Clone this repository to any directory and run

sudo -Hu git make deploy

Usage

  1. Set the login shell of git to /usr/bin/git-shell
  2. Login as git via SSH
  3. You will see a list of commands. Just follow the instructions printed
